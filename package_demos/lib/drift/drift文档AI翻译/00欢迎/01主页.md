# 欢迎使用 Drift！

Drift 是一款面向 Dart 和 Flutter 应用、用于**关系型数据**的响应式本地持久化数据库库。

Drift 是适用于 Dart、Flutter 应用的响应式持久化库。它基于 sqlite3、sqflite 等底层原生数据库库封装构建。在底层数据库能力之上，Drift 额外提供了以下核心特性：

- **类型安全**：你无需手动编写原生 SQL 查询，也无需解析查询返回的动态字典列表 `List<Map<String, dynamic>>`；Drift 会自动将数据库行数据，转换为你自定义的 Dart 对象。
- **流式查询**：你可以零额外成本「监听（watch）」任意查询。任意一条查询语句都能转为**自动更新的数据流**，当底层数据发生变更时，会自动推送最新数据。
- **简洁链式查询**：Drift 会自动生成 Dart 方法与类，让你用 Dart 代码编写查询、自动获取解析完成的结果。
  只需一行 `select(users).watch()`，就能持有一份实时更新的用户列表。就是这么简单！无需手写 SQL，无需手动解析行数据。
- **类型安全原生 SQL 支持**：如果你习惯手写原生 SQL，完全可以继续使用！Drift 内置 SQL 解析器与静态分析器。它能在编译期解析你的 SQL 语句、自动识别返回字段，并生成对应 Dart 代码来承载行数据。
- **数据库迁移工具**：Drift 提供 `.createAllTables()` 等工具方法，极大简化数据库版本迁移的编写工作。你无需手动编写、持续维护所有 `CREATE TABLE` 建表语句。

除此之外还有大量实用能力！
Drift 会在数据插入前做合法性校验，你能拿到友好易懂的错误提示，而不是冰冷的 SQL 错误码。同时它原生支持**事务、DAO 数据访问层、高效批量插入语句**等能力，功能持续完善。
