# Flutter 学习项目

跟随 [Flutter 官方教程](https://docs.flutter.cn/learn/pathway/tutorial) 学习 Flutter 开发时编写的教程小项目和 Demo。

## 已完成项目

这些项目来自 Flutter 官方教程系列，涵盖了 Flutter 开发的核心概念。

| 项目                                                              | 说明           | 学习重点                                      |
| ----------------------------------------------------------------- | -------------- | --------------------------------------------- |
| [birdle](./flutter_official_tutorials/birdle)                     | 猜词游戏       | Widget 基础、用户输入处理、状态管理、隐式动画 |
| [rolodex](./flutter_official_tutorials/rolodex)                   | 联系人列表     | 自适应布局、滚动与 Slivers、堆栈导航          |
| [wikipedia_reader](./flutter_official_tutorials/wikipedia_reader) | 维基百科阅读器 | HTTP 请求、ChangeNotifier、ListenableBuilder  |

## 环境要求

- Flutter SDK
- Dart SDK

## 运行项目

```bash
cd flutter_official_tutorials/<project_name>
flutter pub get
flutter run
```
