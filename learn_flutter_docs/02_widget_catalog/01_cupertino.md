# Cupertino 风格常用组件

| 类名                                       | 描述                                                 |
| ------------------------------------------ | ---------------------------------------------------- |
| CupertinoActionSheet                       | iOS 风格底部弹出动作表单，提供一组操作选项供用户选择 |
| CupertinoActionSheetAction                 | 配合 CupertinoActionSheet 使用的操作按钮             |
| CupertinoActivityIndicator                 | iOS 风格环形加载动画（菊花加载）                     |
| CupertinoAdaptiveTextSelectionToolbar      | 自适应的 iOS 风格文本选择工具栏                      |
| CupertinoAlertDialog                       | iOS 风格警告 / 确认弹窗                              |
| CupertinoApp                               | 应用根组件，配置 iOS 风格全局主题与路由              |
| CupertinoButton                            | iOS 风格扁平按钮                                     |
| CupertinoCheckbox                          | 适配 macOS 风格的复选框（iOS 场景较少用）            |
| CupertinoColors                            | 预定义 iOS 系统标准色值集合                          |
| CupertinoContextMenu                       | iOS 风格长按弹出上下文菜单                           |
| CupertinoContextMenuAction                 | 上下文菜单中的操作项                                 |
| CupertinoDatePicker                        | iOS 滚轮式日期选择器                                 |
| CupertinoDesktopTextSelectionControls      | 桌面端 macOS 风格文本选择控制组件                    |
| CupertinoDesktopTextSelectionToolbarButton | 桌面端 macOS 风格文本选择工具栏按钮                  |
| CupertinoDialogAction                      | iOS 风格对话框的操作按钮                             |
| CupertinoDialogRoute                       | iOS 风格对话框的路由                                 |
| CupertinoDynamicColor                      | 支持深色 / 浅色模式自动切换的 iOS 动态颜色           |
| CupertinoFormRow                           | iOS 风格表单行，用于表单内元素排版                   |
| CupertinoFormSection                       | iOS 风格表单分组，对表单内容做区块划分               |
| CupertinoFullscreenDialogTransition        | iOS 风格全屏对话框的过渡动画                         |
| CupertinoThemeData                         | 存储 iOS 风格主题配置（颜色、字体等）                |
| CupertinoListSection                       | iOS 风格列表分组，带分组标题与分隔样式               |
| CupertinoListTile                          | iOS 风格列表项（单行列表内容）                       |
| CupertinoListTileChevron                   | 列表项右侧的箭头图标，标识可点击跳转                 |
| CupertinoLocalizations                     | Cupertino 组件的本地化字符串资源                     |
| CupertinoMagnifier                         | iOS 风格文本编辑放大镜，显示手指遮挡区域             |
| CupertinoModalPopupRoute                   | 弹出 iOS 风格底部模态弹窗的路由                      |
| CupertinoNavigationBar                     | 标准 iOS 顶部导航栏                                  |
| CupertinoNavigationBarBackButton           | 导航栏内置的 iOS 风格返回按钮                        |
| CupertinoPage                              | iOS 风格的主页面布局组件                             |
| CupertinoPageRoute                         | 带 iOS 侧滑转场效果的页面路由                        |
| CupertinoPageScaffold                      | iOS 风格基础页面结构，包含导航栏与主内容区           |
| CupertinoPageTransition                    | iOS 系统级页面切换转场动画                           |
| CupertinoPicker                            | iOS 滚轮选择器，用于短列表单项选择                   |
| CupertinoPickerDefaultSelectionOverlay     | 滚轮选择器的默认选中项覆盖层                         |
| CupertinoPopupSurface                      | iOS 风格弹窗的表面容器                               |
| CupertinoRadio                             | 适配 macOS 风格的单选框（iOS 场景较少用）            |
| CupertinoScrollBar                         | iOS 风格滚动条                                       |
| CupertinoScrollBehavior                    | 定义 iOS 风格应用的滚动交互行为                      |
| CupertinoSearchTextField                   | iOS 风格搜索输入框                                   |
| CupertinoSlider                            | iOS 风格滑动条，用于数值选择                         |
| CupertinoSlidingSegmentedControl           | iOS 风格可滑动分段选择控件                           |
| CupertinoSliverNavigationBar               | 支持 Sliver 滚动的 iOS 导航栏，可折叠大标题          |
| CupertinoSliverRefreshControl              | iOS 风格下拉刷新组件，适配 Sliver 滚动               |
| CupertinoSpellCheckSuggestionsToolbar      | iOS 风格拼写检查建议工具栏                           |
| CupertinoSwitch                            | iOS 风格开关按钮                                     |
| CupertinoTabBar                            | iOS 风格底部标签栏                                   |
| CupertinoTabController                     | 底部标签栏的状态控制器，管理选中项                   |
| CupertinoTabScaffold                       | 包含底部标签栏的 iOS 主框架结构                      |
| CupertinoTabView                           | 标签栏对应的子页面容器                               |
| CupertinoTextField                         | iOS 风格文本输入框                                   |
| CupertinoTextFormFieldRow                  | 表单内使用的 iOS 风格文本输入行                      |
| CupertinoTextMagnifier                     | 文本编辑专用的 iOS 放大镜                            |
| CupertinoTextSelectionControls             | iOS 风格文本选择控制逻辑                             |
| CupertinoTextSelectionToolbar              | 文本选择时弹出的 iOS 风格工具栏                      |
| CupertinoTextSelectionToolbarButton        | 文本选择工具栏中的按钮                               |
| CupertinoTextThemeData                     | iOS 风格文本主题配置                                 |
| CupertinoTheme                             | 为后代组件提供 iOS 风格主题的作用域组件              |
| CupertinoThumbPainter                      | 绘制 iOS 滑块、开关的拖动按钮（Thumb）               |
| CupertinoTimerPicker                       | iOS 滚轮式时间选择器                                 |
