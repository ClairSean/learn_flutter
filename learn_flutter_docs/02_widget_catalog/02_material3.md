# Material 3 常用组件

## 操作（Actions）

| 组件名                        | 简介                                                         |
| :---------------------------- | :----------------------------------------------------------- |
| Common buttons                | 常用按钮集合，包含 ElevatedButton、FilledButton、FilledButton.tonal、OutlinedButton、TextButton，可点击触发各类操作（如发送邮件、分享文档等） |
| FloatingActionButton          | 包含图标的可点击浮动按钮，始终置于顶层，让关键操作触手可及   |
| Extended FloatingActionButton | 较普通浮动按钮更宽，可容纳文本标签，提供更大的点击区域       |
| IconButton                    | 可点击图标，用于提示用户执行补充操作（supplementary actions） |
| SegmentedButton               | 单个或多个可选中的可点击块，帮助用户选择选项、切换视图或排序元素 |

## 交流（Communication）

| 组件名                  | 简介                                                       |
| :---------------------- | :--------------------------------------------------------- |
| Badge                   | 类图标块，用于传递计数、状态等动态内容，可包含标签或数字   |
| LinearProgressIndicator | 线性进度指示器，随应用加载、表单提交等进程完成而变色       |
| SnackBar                | 简短提示条，在屏幕底部弹出，用于展示应用进程相关的临时消息 |

## 容器（Containment）

| 组件名      | 简介                                                 |
| :---------- | :--------------------------------------------------- |
| AlertDialog | 悬浮弹窗容器，用于提示用户提供更多信息或做出决策     |
| BottomSheet | 锚定在屏幕底部的容器，用于展示补充内容               |
| Card        | 带圆角和阴影的容器，用于展示简短、相关的内容块       |
| Divider     | 细线组件，用于在列表、容器中分组内容，区分不同模块   |
| ListTile    | 固定高度的单行组件，通常包含文本，以及头部或尾部图标 |

## 导航（Navigation）

| 组件名           | 简介                                                     |
| :--------------- | :------------------------------------------------------- |
| AppBar           | 位于屏幕顶部的容器，用于显示页面内容和操作按钮           |
| BottomAppBar     | 位于屏幕底部的容器，用于展示导航选项和关键操作           |
| NavigationBar    | 持久化导航容器，用于在应用中切换主要页面或功能模块       |
| NavigationDrawer | 从应用左侧滑出的容器，用于导航到应用的其他章节或功能     |
| NavigationRail   | 适配平板、桌面端的持久化容器，位于应用左侧，用于导航操作 |
| TabBar           | 分层容器，用于跨不同屏幕、数据集组织内容，支持切换交互   |

## 选择（Selection）

| 组件名     | 简介                                                        |
| :--------- | :---------------------------------------------------------- |
| Checkbox   | 表单控件，用户可勾选或取消，用于从一组选项中选择一个或多个  |
| Chip       | 小型功能块，用于简化信息输入、筛选内容或触发辅助操作        |
| DatePicker | 日历样式选择器，用于选择单个日期或日期范围                  |
| Menu       | 临时弹窗容器，用于展示一系列可选择的操作选项                |
| Radio      | 表单控件，用户可勾选或取消，用于从一组选项中选择唯一一项    |
| Slider     | 滑动选择控件，用于在指定范围内选择一个数值                  |
| Switch     | 切换控件，用于将单个项目的状态切换为开启（on）或关闭（off） |
| TimePicker | 时钟样式选择器，用于选择并设置具体时间                      |

## 文本输入（Text Inputs）

| 组件名    | 简介                                                       |
| :-------- | :--------------------------------------------------------- |
| TextField | 文本输入框，用于在表单、对话框等场景中，让用户输入文本内容 |
