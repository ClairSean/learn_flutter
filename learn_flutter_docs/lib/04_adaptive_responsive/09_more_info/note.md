## 学习基础可用性原则

本章节并未列出所有需要考量的内容。应用支持的操作系统、设备形态与输入设备越多，设计中需要明确的各种组合规则就越复杂。

作为开发者，花时间学习基础可用性原则，能帮你做出更合理的决策，减少开发过程中与设计的反复沟通迭代，最终提升开发效率、获得更好的产品效果。
